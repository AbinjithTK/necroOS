# Product Overview

NecroOS is a browser-based horror-themed Windows 95 operating system simulation. The application progressively becomes more unsettling as users interact with it, featuring glitch effects, ambient audio, jump scares, and a corrupted assistant character.

## Core Concept

- Nostalgic Windows 95 aesthetic using react95 component library
- Progressive horror through a "haunt level" system that increases with user interaction
- Interactive desktop environment with windows, applications, and system features
- Audio-visual effects that intensify as the haunt level rises

## Key Features

- Desktop environment with draggable, resizable windows
- Multiple applications: Notepad, Minesweeper, Portfolio, Terminal, My Corpse, README, Dark Web
- CRT monitor effects and visual glitches
- Ambient audio and sound effects
- Jump scares triggered by haunt level thresholds
- Corrupted Clippy assistant that appears with unsettling messages
